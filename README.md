# TaskFlow - Plataforma de Gestión de Tareas Colaborativas

Aplicación full stack para gestionar tareas en equipo con notificaciones en tiempo real.

## 🚀 Tecnologías
- **Frontend:** React + Vite
- **Backend:** Python + Flask
- **Base de datos:** PostgreSQL
- **Tiempo real:** Socket.io
